## 4. Out-of-Scope Boundaries

VP-Flow v2.0 will NOT include:

- **External user access** — System remains internal only
- **Public-facing portals** — No citizen/public interfaces
- **Multi-tenant architecture** — Single office deployment
- **Third-party calendar sync** — VP-Flow remains system of record
- **AI/ML features** — No automated decision-making
- **Chat/messaging** — Not a communication platform
- **Financial/budget tracking** — Not a financial system
- **HR management** — Not an HR system

# VP-Flow Training Checklist
**Version:** 1.3  
**Effective Date:** 2026-01-26  
**Classification:** Internal Use Only  
**Authority:** Office of the Vice President

---

## Purpose

This document provides structured training checklists for onboarding new staff members to VP-Flow. Each role has a dedicated section with competency requirements and sign-off blocks for verification.

---

## Instructions

**For Trainers:**
1. Review all checklist items with trainee
2. Demonstrate each task in the live system
3. Observe trainee performing each task
4. Check off items only after successful demonstration
5. Complete sign-off block upon module completion

**For Trainees:**
1. Follow trainer through each module
2. Practice each task until comfortable
3. Ask questions before signing off
4. Keep copy of completed checklist for reference

**Completion Criteria:**
- All checkbox items marked complete
- All signature blocks signed and dated
- VP acknowledgment obtained

# VP-Flow v2.0 — Restore Points
This directory contains all restore points for v2.0 development phases.

## Naming Convention
- `RP-v2.0-[phase-name]-PRE.md` — Before execution
- `RP-v2.0-[phase-name]-POST.md` — After execution

## Rules
- v1.x restore points MUST NOT be modified
- Every v2.0 phase requires PRE and POST restore points
